# Contenido Legal

Esta carpeta contiene todos los documentos legales del sitio web de Fruco en formato Markdown.

## Estructura

```
legal/
├── es/          # Documentos en español
├── en/          # Documentos en inglés
├── fr/          # Documentos en francés
└── pt/          # Documentos en portugués
```

## Documentos Disponibles

Cada carpeta de idioma contiene los siguientes documentos:

1. **aviso-legal.md** - Aviso Legal / Legal Notice
2. **politica-cookies.md** - Política de Cookies / Cookie Policy
3. **politica-privacidad.md** - Política de Privacidad / Privacy Policy
4. **politica-calidad.md** - Política de Calidad / Quality Policy
5. **politica-confidencialidad.md** - Política de Confidencialidad y Propiedad Industrial e Intelectual
6. **canal-etico.md** - Canal Ético / Ethics Channel
7. **codigo-conducta.md** - Código de Conducta / Code of Conduct

## Cómo Editar

Los documentos están en formato Markdown. Para editarlos:

1. Navega a la carpeta del idioma correspondiente (es, en, fr, pt)
2. Abre el archivo .md que deseas editar
3. Escribe o pega el contenido en Markdown
4. Guarda el archivo

### Formato Markdown Soportado

Los documentos soportan los siguientes elementos de Markdown:

- `# Título Principal` - Para el título principal
- `## Sección` - Para secciones
- `### Subsección` - Para subsecciones
- `**negrita**` - Para texto en negrita
- `*cursiva*` - Para texto en cursiva
- Párrafos separados por líneas en blanco

## Acceso a los Documentos

Los documentos legales son accesibles desde el footer del sitio web en todas las páginas.

Las URLs siguen el patrón:

- `/legal/es/aviso-legal` - Español
- `/legal/en/aviso-legal` - Inglés
- `/legal/fr/aviso-legal` - Francés
- `/legal/pt/aviso-legal` - Portugués

## Notas

- Los archivos están preparados y listos para recibir contenido
- Los comentarios HTML `<!-- -->` son opcionales y pueden ser eliminados
- El sistema de traducción está completamente implementado
- Los enlaces en el footer se generan automáticamente
